# Checklist de Publicação

Use este checklist antes de mandar o link para alguém.

## Conferir telas

- Abra `index.html` e teste um agendamento.
- Abra `painel.html`, entre com a senha `1234` e veja se o agendamento aparece.
- Clique em `Gerar demonstração` no painel para mostrar dados preenchidos.
- Abra `configuracoes.html` e confirme se serviços, horários e WhatsApp estão corretos.

## Conferir dados comerciais

- Troque o nome `BrilhoMax` se for apresentar para um lava rápido específico.
- Coloque o WhatsApp real.
- Ajuste endereço, serviços, preços, duração e quantidade de boxes.
- Exporte um backup pelo painel antes de fazer muitos testes.

## Publicar

- Crie um repositório no GitHub.
- Envie os arquivos da raiz do projeto.
- Ative GitHub Pages em `Settings > Pages`.
- Escolha branch `main` e pasta `/root`.
- Aguarde o GitHub gerar o link.

## Depois de publicar

- Abra o link no computador.
- Abra o link no celular.
- Teste a tela do cliente.
- Teste login no painel.
- Lembre que os dados ainda ficam no navegador de cada aparelho.
